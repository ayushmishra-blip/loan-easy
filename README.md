# loan-easy
- Allow scripts to run in powershell
    ```
    Set-ExecutionPolicy RemoteSigned
    ```
- Install git
- Install nvm
- Run:
    ```
    nvm install latest
    ```
    ```
    nvm use latest
    ```
- Run:
    ```
    git config --global user.email "you@example.com"
    git config --global user.name "Your Name"
    ```
